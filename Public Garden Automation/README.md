# Public Garden Automation System

## Overview
This project automates irrigation and lighting in a public garden using sensor-based control logic.

## System Features
- Automatic watering based on soil moisture level
- Automatic lighting based on ambient light conditions
- Reduces water and energy wastage

## Sensors & Components
- Soil moisture sensor
- LDR (light sensor)
- Raspberry Pi Pico
- Motor driver
- Water pump
- Lighting system

## Software
- Platform: Arduino IDE (Pico core)
- Language: Embedded C / C++
