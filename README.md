# Robotics Projects

This repository contains academic and self-learning robotics and automation projects developed during my undergraduate studies.  
The projects focus on embedded systems, autonomous robots, basic computer vision, and automation concepts implemented through hands-on work.

---

## Projects Included

### 1. AGRI-SCOUT – Autonomous Agricultural Rover *(In Progress)*
- 6-wheel drive autonomous rover designed for agricultural fields
- GPS-free navigation between crop rows using LiDAR
- Camera and sensor-based analysis for monitoring crop conditions
- Focus areas: Autonomous navigation, sensor integration, agricultural automation

---

### 2. Autonomous Vehicle and Smart Traffic Signal System (V2I)
- Prototype demonstrating communication between a traffic signal and a vehicle
- Dual ESP32 setup for wireless data transmission
- Vehicle motion controlled based on traffic signal status
- Focus areas: Embedded systems, wireless communication, autonomous control logic

---

### 3. Maze Solving Robot using Dijkstra’s Algorithm
- Robot navigates a maze using shortest path calculation
- Maze modeled as nodes and paths for decision making
- Smooth movement achieved using basic PID control
- Focus areas: Algorithms, path planning, robot control

---

### 4. Inventory Control Robot
- Robot designed to assist with inventory tracking and warehouse tasks
- Uses a camera module for detecting and counting items
- Wireless communication for sending inventory data
- Focus areas: Automation, embedded systems, system integration

---

### 5. Food Sorting System using Shape Detection
- Automated system for sorting food items on a conveyor
- Uses shape analysis to identify defective or broken items
- Implemented using OpenCV on a Raspberry Pi
- Focus areas: Image processing basics, industrial automation

---

### 6. Public Garden Automation System
- Automated irrigation and lighting system for public gardens
- Soil moisture-based water control
- Light-based control for garden lighting
- Focus areas: Automation, sensor-based control, sustainable systems

---

## Technologies & Tools Used
- Embedded C / Arduino
- ESP32, Raspberry Pi Pico, Raspberry Pi, Pi Cam Module
- Sensors, motor drivers, DC motors
- Python (OpenCV basics)
- Robotics and automation fundamentals

All projects are academic, prototype-level implementations developed as part of coursework and self-learning.
