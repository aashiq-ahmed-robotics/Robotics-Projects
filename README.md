# Robotics Projects

This repository contains academic and self-learning robotics and automation projects developed during my undergraduate studies.  
The projects focus on embedded systems, autonomous navigation, computer vision, and automation concepts.

---

## Projects Included

### 1. AGRI-SCOUT – Autonomous Agricultural Rover *(In Progress)*
- 6WD autonomous rover for crop health monitoring
- GPS-free LiDAR-based row (tunnel) navigation
- Virtual sensing using computer vision and signal processing
- Focus areas: Autonomous navigation, virtual sensors, precision agriculture

---

### 2. Autonomous Vehicle and Smart Traffic Signal System (V2I)
- Vehicle-to-Infrastructure (V2I) prototype using dual ESP32
- Wireless communication between traffic signal and autonomous vehicle
- Vehicle motion controlled based on real-time signal state
- Focus areas: Embedded systems, wireless communication, autonomous decision logic

---

### 3. Maze Solving Robot using Dijkstra’s Algorithm
- Autonomous robot navigation using shortest path algorithm
- Maze modeled as a graph with nodes and edges
- Logical path planning with PID-based motor control
- Focus areas: Algorithms, path planning, control systems

---

### 4. Inventory Control Robot
- Embedded robot for inventory tracking and warehouse automation
- Object detection using camera module
- Wireless data transmission for inventory monitoring
- Focus areas: Automation, embedded systems, system integration

---

### 5. Food Sorting System using Shape Detection
- Automated food sorting using computer vision
- Shape-based defect detection using OpenCV
- Real-time processing on conveyor setup
- Focus areas: Image processing, automation, industrial applications

---

### 6. Public Garden Automation System
- Automated irrigation and lighting control system
- Soil moisture-based watering
- Light-dependent lighting control
- Focus areas: IoT, automation, sustainable systems

---

## Technologies & Tools
- Embedded C / Arduino
- ESP32, Raspberry Pi Pico, Raspberry Pi, Pi Cam Module
- Sensors, motor drivers, actuators
- Python (OpenCV)
- Robotics and automation fundamentals

All projects are prototype-level academic implementations developed for learning and experimentation purposes.
